# 📋 TAREA 4: Selección de Entradas y Simulación de Pagos

## ✅ Completado

Esta tarea incluye la implementación completa de:

### 🛒 **Carrito de Compra**
- Seleccionar cantidad de entradas
- Validar disponibilidad de stock
- Cálculo automático de subtotal, impuesto y total

### 💳 **Procesamiento de Pagos**
- Simulación de pagos (95% éxito, 5% rechazo)
- Validación de tarjeta de crédito/débito
- Generación de número de transacción único

### 🎫 **Generación de Entradas Digitales**
- Código QR único por entrada
- Estado de entrada (ACTIVA, USADA, CANCELADA)
- Vinculación usuario-evento

### 📊 **Gestión de Reportes**
- Historial de compras por usuario
- Ingresos totales del sistema
- Entradas vendidas por evento
